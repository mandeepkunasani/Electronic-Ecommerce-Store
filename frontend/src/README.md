# Electronic Ecommerce Store

A full-stack Ecommerce Web Application built using the MERN Stack (MongoDB, Express.js, React.js, Node.js).

## Features

### User Authentication
- User Registration
- User Login
- User Logout
- MongoDB User Storage

### Product Management
- Product Listing
- Product Search
- Category Filtering
- Product Details Popup

### Shopping Features
- Add To Cart
- Update Quantity
- Remove Products
- Wishlist Management

### Checkout System
- Checkout Form
- Order Placement
- Order History Tracking

### Reviews
- Product Reviews
- Star Ratings
- Review Storage using Local Storage

### Additional Features
- Responsive UI
- Dark Theme Design
- Contact Us Section

## Installation

### Clone Repository

git clone https://github.com/yourusername/electronic-ecommerce-store.git

### Install Frontend

cd frontend
npm install
npm run dev

### Install Backend

cd backend
npm install
npm start
